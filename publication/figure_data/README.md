# Figure Source Tables

This directory contains compact source tables for the public figure assets.

- `figure2_domain_summary.csv`: model-level Coverage, Fidelity, and Discriminability values used for the main domain summary.
- `figure3_recovery_profile.csv`: edge-recovery decomposition used for the recovery-profile display.
- `figure4_family_heatmap.csv`: family-stratified metric table used for the family heatmap.
- `supplementary_kg_overlay_source.csv`: gold-edge and model-recovery source table for the supplementary dense KG overlay.
- `supplementary_threshold_sensitivity.csv`: threshold-sensitivity source table.
- `supplementary_run_count_sensitivity_summary.csv`: run-count sensitivity summary source table.
- `supplementary_bootstrap_ci_summary.csv`: bootstrap confidence-interval source table.
- `supplementary_risk_weighted_fidelity.csv`: risk-weighted fidelity source table.

The full machine-readable outputs are retained under `results/`.
