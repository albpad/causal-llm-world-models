# Publication Assets

This directory contains non-manuscript assets needed to audit or reproduce the article displays.

## Main Figures

- `figures/main/figure1_method_schematic.*`: schematic of KG1 construction, vignette perturbation, KG2 recovery, and domain-based evaluation.
- `figures/main/figure2_domain_summary.*`: six-model Coverage, Fidelity, and Discriminability summary.
- `figures/main/figure3_recovery_profile.*`: edge-recovery profile separating correct, wrong-direction, and unrecovered edges.
- `figures/main/figure4_family_heatmap.*`: family-stratified heterogeneity summary; `cT4a unselected` is marked `NA` because it has 0 gold evaluation edges.

## Supplementary Figures

- `figures/supplementary/supplementary_figure1_kg_overlay.*`: dense KG overlay retained for auditability.
- `figures/supplementary/supplementary_figureS1_threshold_sensitivity.*`: threshold-sensitivity analysis.
- `figures/supplementary/supplementary_figureS2_run_count_sensitivity.*`: run-count sensitivity analysis.

## Source Tables

The `figure_data/` directory contains compact figure-source tables. The full analysis artifacts remain under `results/`.

Manuscript `.docx` files and prior manuscript versions are intentionally excluded from this repository.
