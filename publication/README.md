# Publication Assets

This directory contains the final non-manuscript publication assets referenced by the cleaned repository:

- `figure_data/`: source tables used to derive the final publication figures
- `figures/main/`: Figures 1 to 4 in `PNG` and `SVG`
- `figures/supplementary/`: supplementary threshold-sensitivity figure assets

These assets are retained for auditability and journal production support. The manuscript text and document files are intentionally excluded from the repository.
